# 词汇解释

## 基本概念

<a id="test_object"></a>
### {{book.test_object}}
模型中包含了某个控件的属性集合信息，可以用于定位控件。

<a id="identification_properties"></a>

### 识别属性
用来识别某个控件的属性。一个{{book.test_object}}可以有一个或多个属性。所有属性集合在一起，用“And”的方式来识别控件。这意味着如果一个识别属性不匹配，就无法识别控件。

<a id="index_properties"></a>

### Index属性
作为一个特殊的属性，Index属性可以在其它识别属性不能唯一标识控件时，用作索引定位其中的一个控件。


<a id="object_model"></a>

### 对象模型
以*.tmodel后缀结尾的文件，包含一组{{book.test_object}}和他们之间关系的信息，对象模型用来管理应用程序的控件识别信息。当一个应用程序更新后，通常需要更新对象模型，使对象模型保持同步，更新的工作需要用户手动完成。
