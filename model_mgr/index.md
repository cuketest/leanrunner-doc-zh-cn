# 模型管理器


模型管理器(Model Manager)用来管理桌面应用控件对象的树状结构，它存储了各种类型的{{book.test_object}}，用于识别控件，以及截获控件的快照等。模型管理器可以对{{book.test_object}}的信息做各种编辑操作。
